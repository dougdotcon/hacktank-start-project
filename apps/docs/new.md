# Modalidades de Hackathons no HackTank

## 1. Hackathon Patrocinado por Empresa

### Característica Principal
Uma empresa busca desenvolver uma solução específica e oferece prêmios para os vencedores.

### Modelos de Premiação
- **Modelo Compartilhado**: R$ 4.000 para o vencedor + participação para continuar desenvolvendo a ideia
- **Modelo Aquisição Total**: R$ 15.000 para o vencedor, com a propriedade intelectual transferida para a empresa via contrato formal

## 2. Hackathon Colaborativo de Investimento

### Característica Principal
Grupo de investidores aportam capital em conjunto e o MVP vencedor recebe o investimento acumulado.

### Estrutura de Participação
- Todos os participantes recebem 20% de participação distribuída igualmente
- Possibilidade de organização posterior para trabalho em equipe

## 3. Hackathon "Investidor-Desenvolvedor"

### Característica Principal
Um investidor aporta capital (ex: R$ 2.000) para atrair desenvolvedores que criarão uma solução.

### Funcionamento
- 5 desenvolvedores competem para criar a melhor solução
- O investidor escolhe o vencedor que recebe o prêmio
- O vencedor continua trabalhando com o investidor para desenvolver a ideia

### Vantagens
- Para o investidor: Obtém um MVP validado com baixo custo
- Para os desenvolvedores: Oportunidade de ganho rápido e portfolio

### Características Gerais
- Prazo típico: 48 horas (formato hackathon padrão)
- Benefício mútuo para investidores e desenvolvedores
- Validação rápida de ideias com protótipos funcionais

### Exemplo de Caso
Investidor deseja criar um bot para trading de criptomoedas:
- Paga um valor acessível para obter um MVP validado
- Obtém resultados rápidos sem compromissos longos
- Desenvolvedores ganham experiência e remuneração imediata
